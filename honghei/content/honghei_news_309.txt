首先不得不提下诺基亚Asha平台，该系统平台由诺基亚基于2012年收购Smarterphone研发的智能手机移动OS，并于2013年发布首款Asha平台设备，即诺基亚501，主打入门，面向新用户。

从全球手机市场来看，安卓系统无疑是最大移动OS平台，用户数量更是惊人，相比安卓，小众的Asha平台更显冷门，在入门安卓设备的攻势下，节节败退。

另外一方面，这批Nokia X系列廉价安卓设备市场反响异常火爆。
安卓老用户继续使用熟悉的系统平台，新用户也可以低预算购买Nokia X设备。
但诺基亚Nokia X并不是一款微软系列设备，其搭载使用了竞争对手谷歌的安卓系统，因此，对于微软来说，诺基亚Nokia X过于成功也会影响其市场策略。
之前，外媒分析认为主流电脑手机厂商将试水Windows8+Android、Windows Phone+Android双系统移动设备，但均被谷歌、微软阻止。
是继续增加Nokia X系列设备？还是好面子，重新启用Asha，取消Nokia X产品？
这个“二选一”艰难的决定最终还得由微软敲定。

苹果公司创始人史蒂芬乔布斯曾为 Linux 和 Git 这两个伟大的开源项目创始人托瓦兹提供工作。托瓦兹从未见过微软公司的比尔盖茨，但却在2000年的时候遇上了乔布斯。那时候的他任职于美国半导体设计公司全美达（Transmeta）。乔布斯邀请托瓦兹去位于库比蒂诺的苹果总部工作，为他提供丰厚的薪水和优越的职位，但是他在苹果的工作却是与Linux 无关的，正是由于这个原因托瓦兹拒绝了乔布斯的邀请。

况且，托瓦兹根本就不喜欢苹果操作系统的内核Mach。

如果托瓦兹接受了邀请会怎么样？
在那个时候如果托瓦兹答应乔布斯去苹果工作，那现在的世界将会不一样。我们不会有 Linux，Kindle 和 Android。互联网上一半以上的内容和超过九成的服务器都将不复存在。毫不夸张的说，世界绝不可能是现在这个样子。
托瓦兹拒绝乔布斯的负面影响有哪些？
原本我们可以看到托瓦兹和乔布斯两个天才在一起工作。或许世界也将从他们俩的合作中获益。但可以肯定的是，我们将冒着失去 Linux 的风险。
托瓦兹为自己代言
有时候，托瓦兹由于他那些极具争议的言论而显得傲慢。但这与他对世界的贡献是不相符的。他创造了 Linux 和 Git 两个极好的开源项目，并使他们自由免费。有许多攻讦托瓦兹的言论说他从没有拿出他的部分收入来改善这个世界。但事实上，托瓦兹对于他人贡献是难以想象的。托瓦兹将他最完美的作品变成免费。而他本可以从中获得数以亿计的收入。他将他所拥有的一切无私平等地奉献给了整个世界。
Linux 内核的总设计师是一个很好又有趣的人。他谈论着世界的各个方面。他的博客涉及孩子、万圣节和祖国等方方面面。你可以通过博客和 G+ 来关注他。

     [03.25]为何中国没有自主操作系统？ 

我国互联网产业发展存在一定的薄弱环节，例如在产业布局上，作为IT基础设施的操作系统一直没有实现国产化，国内企业在此领域一直缺位。对此，倪光南院士做客《科技不怕问》栏目，回答网友相关提问。网友“赶考书生”：操作系统在整个IT产业的重要性就不用说了，可是这么多年发展下来，为什么中国没有强大的自主操作系统，依然受制于人？腾讯科技：我国也有企业或事业单位尝试推出国产自有操作系统，例如今年年初，同洲电子推出了号称国内首套自主研发的“960”手机操作系统，中科院软件研究所也联合上海联彤公司发布了一款所谓自主智能手机操作系统：COS。不过不论是用户数、应用数量，还是终端覆盖率，这些操作系统都还不构成市场影响力。那么原因何在？=700) window.open('http://img1.gtimg.com/tech/pics/hv1/103/63/1552/100934968.jpg')" style="max-width:700pxmax-height:700px" onload="if(is_ie6&&this.offsetWidth>700)this.width=700" >倪光南：操作系统分为不同类别，具体情况有所不同。在云计算环境下的操作系统主要可分成三类：一种是云计算中心使用的操作系统，这类操作系统支撑着各种信息系统和网络系统；一种是各种智能终端使用的操作系统，用于各种PC、笔记本、超级本等桌面终端，智能手机、平板电脑等移动终端，机顶盒、智能电视等各种家用终端，还有可穿戴设备（手表、眼镜）和车载电脑等等；此外还有嵌入式操作系统，用于工控设备、物联网设备和消费电子设备等等。第一类操作系统方面，国产系统的发展不错，例如中国的因特网企业、高性能计算机等使用的系统大多是国产的，第二三类系统的国产化情况很差。尤其是智能终端操作系统，目前全世界基本上只有三家――苹果、谷歌(微博)和微软，中国也不例外。智能终端操作系统具有高度垄断性，也是信息领域生态系统的核心，上述这三家，它们是世界最大的三家IT企业，可见需要很强的实力才能成为这个领域的一员。目前中国技术实力最强的华为，虽然在通信设备领域已是世界第一，但相对于这三家而言，还有一定差距，例如华为市值估计是1200亿美元左右，大概是这三家公司的1/3左右。中国移动([backcolor=url(http://mat1.gtimg.com/digi/dc_new/icon_wb_ly.gif) 4px 3px no-repeat]微博)等企业似乎市值不小，但研发实力也仍有不小差距，更多的企业是大而不强，所以要做智能终端操作系统还需要注重整合资源。过去我国科技规划对此也有支持，但缺乏顶层设计。以往只支持几家小公司做桌面操作系统，在知识产权问题缺乏研究的情况下，支持多家在国外移动系统上修改，做移动操作系统，形成低水平重复。另外在总体计划安排上，对这类操作系统的垄断性缺乏认识，往往只注意某些特定应用，未重视公众服务市场。这些因素都造成我国在这个领域长期受制于人。
