     [04.12]SQL Server2014 改变数据库的游戏规则 

在大掌控、大智汇、大洞察这样的主题下，微软大数据战略支柱的角色落在了新的产品Microsoft SQL Server2014上，同时还包括了微软的Microsoft Azure、Microsoft Server、Microsoft Office。可见，微软的战略触及用户数据访问、BI工具、统一数据平台。
　　4月10日，微软全球数据平台业务总监Jack Tang表示，希望用户通过Microsoft SQL Server2014可以获得突破式的高性能平台。
　　官方透露的微软Cloud OS愿景：实现现代化企业统一平台。即：转换数据中心、释放数据洞察力、促进以人为中心的IT、提供现代化业务应用。
　　微软数据平台进化从Microsoft SQL Server2000到Microsoft SQL Server2014，已经将主题锁定在云与关键业务。并且，新的版本对于用户关键业务CRM等有着性能的提升。同时：
　　Microsoft SQL Server2014内置内存技术，提升了应用速度。
　　Microsoft SQL Server2014在安全性与扩展性方面有所考量。
　　Microsoft SQL Server2014在高可用方面继续延续ALWAYSON优势。
　　随着内存成本走低，CPU整体表现基本持平。因此，在这样的前提下，Microsoft SQL Server2014的出现，对内置内存技术有了新诉求和创新。某大型零商的SAAS提供商将其应用放在内存里面，性能提升7倍。也就是采用了In-Memory内存驻留技术，其实早在Microsoft SQL Server2008R2版时就推出了适用于EXCEL的POWERPIVOT内存驻留技术。
　　在Jack Tang看来，In-Memory内存驻留技术优势在于：
　　一是平均提速10倍，无需重写整个应用，充分利用了Microsoft SQL Server各项功能
　　二是只选择奖利用率高的表放在内存中，对内存技术进行优化，充分适应现有硬件
　　三是针对OLTP\DW以及BI提供高性能的内存技术，所有功能通过单一SKU提供。
　　其主要功能包括了，全新的内存OLTP、针对数据仓库而改善内存列存储技术、通过POWERPIVOT实现内存BI、缓冲池扩展至SSD。
　　进一步分析来看：
　　在安全方面，与业界数据库ORACLE、MYSQL、DB2比较，Microsoft SQL Server是连续5年统计漏洞最少的数据库、全球使用率最高的数据库，目前占有46%市场份额。
　　在扩展性方面，Microsoft SQL Server2014考虑到计算扩展、网络扩展、存储扩展三个方面。针对存储虚拟化，通过标准硬件实现企业级网络存储，通过存储分层提高性能针对网络虚拟化技术提供灵活性与隔离，分配最小和最大带宽具备高达640颗逻辑处理器，每虚拟机64颗VCPU，每虚拟机1T内存每全集64个点。
　　在高可用性方面，改善可用性、提高效率、强调易于部署与管理。
　　在关键业务支持方面，针对整个生态的系统包括7万多 Microsoft SQL Server合作伙伴，“Microsoft SQL Server的DBA通常没有ORACLE的DBA贵。”其中的价值显而易见。
　　据说，“自2012年以来，超过900家企业从其他平台迁移到Microsoft SQL Server。其中金融服务行业占据了96家零散制造业占据70多家。”可见，Microsoft SQL Server已经被重点行业使用到了核心业务系统。
　　从上观点总结来看，Microsoft SQL Server2014强调的是：通过任何数据更快速获得洞察力。“搜索、访问、塑造内部与外部数据，将其与非结构化数据组合在一起通过OFFICE及OFFICE365中熟悉的EXCEL(可以看作前端BI工具)快速分析并获得洞察力，并通过移动设备访问帮助用户扩展BI模型，使用跨越云端和内部环境的完善平台，丰富并保护用户数据。”
　　如何轻松访问或大或小的数据?简化公共数据及企业数据的访问，轻松预览、塑造用户数据，调整数据格式与非结构化数据相结合，跨越不同数据源结合并优化数据，通过关系型、非结构化、及半结构化数据获得洞察力用通用方式管理非结构化数据。
　　如何通过熟悉工具获得更大洞察力?建模与聚合方面在于，访问几乎任何来源的数据进行聚合，使用熟悉的工具进行数据建模采用可视化与分析，通过强大的EXCEL的POWERPIVOT与POWERVIEW结合实现。
　　在后端BI方面，Microsoft SQL Server2014采用了新的功能，STREAMINSIGHT流数据处理技术、DQS数据质量服务、MDS、SSIS分析服务、BI语义模型。
　　进一步分析来看看，Microsoft SQL Server2014的混合云解决方案的市场机会应该在三个方面：
　　一是简化云备份，手工或自动，在实例层面实现时间点还原，通过可衡量的数据库使用模式设置备份频率
　　二是云端灾难恢复，迅速灾难恢复实现
　　三是扩展内部使用。
　　但用户如何更好地迈入云端?微软提供的方式包括：一是新的迁移向导，将内部环境的Microsoft SQL Server迁移到AZURE虚拟机。二是可以通过微软全套的混合云解决方案去实现。
　　由此，在Microsoft SQL Server2014出台后，微软数据平台更为丰富，更接近和使用现有用户对于云的诉求与接入，该平台包括了微软自身的基础架构、数据管理与处理、数据丰富与联合查询、BI与分析的四层架构。好吧，或许，Microsoft SQL Server2014已经开始了改变数据库的游戏规则，不知道友商作何感想?
　　但实际上，早在去年7月左右，面向云的数据库产品Oracle 12c就正式上市了，其中包含了6大特性：全新多租户架构、数据自动优化、深度安全防护、面向数据库云的最大可用性、高效的数据库管理以及简化大数据分析。这些特性可以在高速度、高可扩展、高可靠性和高安全性的数据库平台之上，为客户提供一个全新的多租户架构，用户数据库向云端迁移后可提升企业应用的质量和应用性能，还能降数百个数据库作为一个进行管理，帮助企业迈向云的过程中提高整体运营的灵活性和有效性。
　　只不过，当时的Oracle 12c出台后，不是像Microsoft SQL Server2014这样，很快与合作伙伴(惠普)合作，并推出新一代的并行数据库一体机。Oracle数据库12c刚开始主要与甲骨文SPARC T5服务器实现了相互集成。作为Oracle公有云服务的基础，Oracle 12c可为客户部署私有数据库云，SaaS供应商可以借助Oracle数据库在安全的多租户模型中获取更高价值。
　　可能你会说，Microsoft SQL Server2014采用了内存数据库技术，促进了数据库的效率提升。但是，如果你查询一下资料就会看到，早在2009年Oracle内存数据库TimesTen通过改变数据在运行时驻留位置的假设来提供实时性能。通过在内存中管理数据，并相应地优化数据结构和访问算法，数据库操作能够以最大效率执行，从而大大提高响应速度和吞吐量，甚至能够与完全使用高速缓存磁盘的 RDBMS 相媲美。Oracle内存数据库TimesTen可嵌入到应用程序中，从而消除了进程间通信和不必要的网络操作，以进一步提高数据库操作的性能。
　　可见，Oracle在内存数据库技术、针对云数据库的扩展和安全方面、以及数据库的高速度方面都早已有所部署，因此，在Microsoft SQL Server2014发布之后，如何凭借微软SQL在数据库领域所占据的地位，拓宽在用户转向云过程中可能的市场机会，这将是Microsoft SQL Server2014未来的重点。同时，Microsoft SQL Server2014更不能忽视Oracle这样一个富有攻击性的、有着数据库长期深入积累的同业者。

     [04.12]微软出狠招 “免费”杀shou锏为何故？ 

笔者按：不破不立，既是武侠小说中常出现的桥段，又是中国传统哲学的经典理论之一。用通俗点的话来说，就是不到黄河心不死，不见棺材不掉泪……或许，对于微软来说，这句话也同样适用。在过去几十年微软一直在“摆谱”，突然发现只凭老本混不下去了，才慌忙想起自救的手段来。
正所谓成也萧何，败也萧何，微软是自Windows系统起家，成为用户最为熟知的科技巨头之一，但现在却又因Windows系统离最火爆的移动互联网渐行渐远。微软在移动互联网乃至桌面端的疲软早就暴露于天下，似乎世人很不理解为何这艘超级航空母舰转向为何如此之慢和艰难。幸好，自新CEO上任后，情况有所改变。
微软近来不断发动大招，走出改变的关键几步。而这其中，最受人注目的自然是多项免费措施。鉴于“免费”二字的吸引力之大，鉴于这将颠覆微软一向“雁过拔毛”的收费传统，或许微软的真正变革即将即开，而我们也将看到一个全新的微软。
断腕自救，免费只为抢市场
微软展开变革的措施称得上是“断腕自救”，先是发布iPad版Office套件，首次让iOS用户享受移动办公的乐趣，随后又在近日发布多项免费措施。微软在近日举行的Build 2014大会上正式宣布，将对9英寸以下小型智能手机及平板电脑Windows系统实行免费。再加上新用户将拥有1年的Office 365免费订阅使用、即将问世的物联网Windows也将免费……众多免费措施不仅让业界震惊，也让人们看到微软变革的决心。
9英寸以下的智能终端自然是以智能手机和平板为主，微软此举明显是要针对移动终端市场发力。要知道，微软虽然在移动终端市场排名第三，却远远落后于占据近八成份额的Android和两成份额的iOS系统，仅为5％。除去本身系统不够给力、营销推广不够完善、应用较少等原因，Windows系统在移动终端上不吃香的原因就是在此之前，微软一直秉承“软件应该付费”的理念，向手机和平板电脑厂商按照5-15美元的标淮，对每部设备收取Windows许可费用。这不仅不符合当前移动操作系统免费的趋势，还受到业界的普遍质疑。
如今，微软终于迈出了“非常正确且理智的第一步”，用现任CEO纳德拉的话说，就是“我们将以挑战的心态去创新。我们所要做的，并不是仅局限在对下一个版本的Windows上作出部分创新，未来我们要在方方面面去创新。”
不过，微软显然打着如意算盘。因为目前Windows移动终端市场占有率不高，微软的免费策略并不会为其造成多大损失。而一旦此举有效，不仅能刺激移动终端销售的增长，吸引用户使用微软的移动服务，还能在一定程度上吸引开发者为其开发更多应用，可谓一举两得。
前有狼后有虎，狠招仍需加把劲
很多业内人士都在想，如果微软能提早预知现在的窘境，在四五年前免费，恐怕现在的移动终端市场就不是现在的这幅情景。当初微软的锱铢必较和小家子气既让其坐失良机，又让Android得以壮大，让人扼腕叹息。
虽然微软现在终于幡然醒悟，但晚来的免费策略真的就能让它在移动终端市场从低谷走向巅峰？目前看来，恐怕未必。或许从微软的出发点来看，“我都已经打破传统，让系统免费了，还不让我快点登顶”。这微软想出来的最好办法，却未必适合当前的移动终端市场。要知道，微软面对的可是Android和iOS这两大强劲对手。
先说iOS系统，不论是iPhone还是iPad，苹果都以其独特的封闭模式处于绝对垄断地位。微软凭借免费手段很难俘获果粉的心，也很难在高端移动市场和苹果竞争。或许，放下身价，去抢夺低端市场不失为一个好办法。
微软更想挑战的是谷歌的Android系统，毕竟Android系统就是免费的，而Android系统之所以能够快速崛起，靠的也是免费策略。但在Android系统已经占据大半江山的现在，微软作为追赶者，仅凭同样的免费策略，想要让Windows系统和Android系统平起平坐，无异于白日做梦。
可以说，如果没有特别的功能，没有超越当前其他移动操作系统的创新，没有吸引人的服务，微软的Windows系统想和iOS、Android系统三足鼎立，恐怕只是一场虚无缥缈的梦。免费当然是一个狠招，但还没能彻底地搅乱当前的形势，必须再加把劲，才能让移动终端市场的水“越来越混”，这样微软才正好浑水摸鱼。（

     [04.12]XP系统停止服务怎么办？八成中国用户：死守 


4月8日，微软正式停止对Windows XP操作系统(以下简称“XP系统”)进行技术支持，不再修补XP系统的安全漏洞，此举引起了舆论广泛关注。有评论认为，此举将为微软带来大量收益，但也可能受到更多来自用户的投诉。
零点指标数据公布的网络调查结果显示用户多表示不舍，接近80%的用户称不愿放系XP系统。虽然微软方面提前数月作出提醒，但还是引得部分用户不满，被认为是不负责任，置数亿用户于危险的境地。
近八成用户不愿放弃
根据百度指数分析结果，自去年11月微软宣布XP系统将停止服务后，关于“XP系统停止服务”的舆论反应一直较为平稳，但在今年2月中旬开始，先后出现了两次搜索指数高点。第一次出现在2月14日，最高点达到324663，当时距离XP系统停止服务还有50天，各家媒体第一次集中报道，引发大量网友关注。第二次出现在4月8日，最高点为107169，当日为微软正式对XP系统停止服务，有关的新闻报道也达到了135篇。
在近30天的百度指数方面，“XP系统停止服务”的百度指数同比增长超过1000%。相关的搜索热词排行依次为“XP系统”、“Win7旗舰版”、“XP系统停止服务怎么办”，上升最快的检索词依次为“windowsXP停止服务”、“XP系统停止服务后还能用吗”、“XP系统停止服务怎么办”，三者的上升速率均超过了1000%。
从关注人群来看，92%为男性，近80%集中在20-40岁，关注度最高的省市依次是广东、浙江、江苏和山东。
4月9日，零点指标数据根据其网络调查系统，对北京、上海、广州、成都、青岛和武汉6市的401名18周岁及以上常住居民进行了随机访问，结果有40%用户表示“不会更换系统，将一直继续使用XP系统”；另有37.8%则表示“会先用着XP系统，直到微软不提供任何服务为止”，这表明近八成的用户在目前阶段不愿放弃，将会继续使用XP系统。

习惯使用为不舍主因
稍早些时候，互联网消费调研中心对3267名电脑使用者进行了问卷调查。调查显示，用户对XP系统给予了较高的评价，41.4%的用户表示XP系统是“一款跨时代的产品”，29.4%的用户认为其“让电脑操作更加简单”，XP系统的用户中有80%使用超过五年。
对于XP系统停止服务后的担心，30%的用户选择了“潜在的安全风险”，“兼容性不足”、“操作体验落后”紧随其次。
在表示会继续使用XP系统的用户中，因使用习惯而继续使用的用户占39.8%；23.2%的用户继续使用的原因是电脑配置过低，其他因工作需要等原因而继续用XP系统的用户平均在15%以下。

     [04.12]别了，郑州科技市场 


日前，郑州市政府正式对外发布了《2014年度新型城镇化建设目标任务分解方案》，公布了今年57个要外迁的市场，其中位于河南科技园区(河南科技市场2010年更名为河南科技园区)内的外运大厦、中科大厦、中关大厦、创新大厦、科技市场老市场、新科大厦、电子大厦(淘宝城)等，都将在年底前完成外迁工作。河南科技市场从1993年4月12日开业至今，已经走过了21个年头，那里曾经留下了无数70后、80后、90后淘换电子产品的身影。
【70后】花一万五攒出第一台电脑
“把家里这台旧电脑扔了吧，要不然当废品卖了算了！”4月7日下午，家住郑州市南阳路一小区的张华收拾完房间，再一次对丈夫刘金提出要求：将存放在储物间里的一台发黄的旧电脑清理出去。然而，她得到的回答仍然非常简单：“不扔。”
那台外壳已经发黄的白色电脑虽然十多年前就不能使用了，但刘金一直保存着它。
刘金出生于1974年，1995年大学毕业后就留在了郑州工作，也正是在那一年底，刘金花了一万五千多块钱，在郑州文化路上的科技市场攒出了他的第一台电脑。
“买电脑的钱有一部分是我存了半年的工资，还有部分是父母给的。”刘金告诉记者，这台电脑也是迄今为止他买的最贵的一台，当然配置也最低，内存仅有16M。
“如果当时要买品牌机，要再贵上两三千块钱。”刘金告诉记者，在上个世纪90年代，电脑还是奢侈品，是只有专业科技人员才能使用的高深莫测的东西。“上大学那会儿，我们宿舍几个同学偶尔也会一起趁周末去科技市场转转，当看到CPU、主板、硬盘等零部件时，心中便有一种莫名的兴奋，感觉就是见到了高科技。”刘金称，很长一段时间里，他对河南科技市场都是怀有一份敬畏之情的，因为他对高科技的理解是从那里开始的。
刘金同时表示，其实现在流行的DIY这个词，在他刚工作那会儿就流行了，而主要流行区域就在科技市场，再具体点就是用在攒机上面，“其实就是根据顾客的需求，配置不同性能的电脑”。

【80后】每次去总要淘些电子产品
“电脑、手机、MP3”是不少80后进入大学时必配的三大件。而因为河南科技市场所售各类电子产品比较集中，因此不少80后大学生都会选择到这里淘入学装备。
1985年出生的郭永辉老家在许昌，现在在郑州一家房地产公司工作，他2005年9月从许昌老家来到郑州上大学。开学前一天，父亲带着他到文化路上的河南科技市场转了半天，买了一台电脑、一部手机和一个MP3。“当时记得买这些东西花了我爸7000多块钱，电脑买的是组装机，花了将近4000块钱，手机是诺基亚的。”
郭永辉告诉记者，他当时买的诺基亚6681摄像头是130万像素的，内存8MB，“当时已经非常智能了，在其他地方都得卖4000多块钱，但在科技市场只花了3500多块钱。”那是他第一次到河南科技市场去，之后便会经常跑去那里淘宝，大学四年，他换了3部手机，都是从科技市场买的，很便宜。
“感觉科技市场就像一个百宝箱，只要是想找的科技产品，都能从那里淘到。”郭永辉说，他是他们宿舍8个人中第一个配电脑的，后来宿舍其他同学配电脑、买MP3和手机啥的，也都是他领着到科技市场挑的。
已经参加工作多年的郭永辉表示，现在他没事也会到科技市场转转，每次去都会淘一些电子产品。“那里经常会推出一些比较新的电子产品，去年的时候郑州市开始对电视机顶盒进行更新，我到科技市场转悠，就看到卖的有那种无线网络机顶盒，花了200多块钱买了一个”。
【90后】到科技市场刷智能手机
小冯是郑州一高校的大二学生，标准的90后，他告诉记者，他们学校虽然就在郑州市区，但他和同学们并不经常到科技市场去。“感觉太乱，每次从那里经过，都会有不少人围上来问要不要配电脑，还老堵车”。
小冯说，对于不少90后学生来说，玩是他们的一大爱好，其中就包括手机，“现在大家用的都是智能手机，除了上课和睡觉，手机几乎都是不离手的”。
小冯用的是一部三星Note2，2012年花四千多块钱买的，去年下半年开始经常出现死机现象。由于过了保修期，到三星售后去刷机的话，每次需要50块钱。今年年初，小冯来到科技市场，找了一个手机维修点，花了20块钱便完成了刷机。
“我们班不少同学刷机都是到科技市场，便宜一些。”小冯表示，除了这次刷机，去年10月刚升大二的他，由于经常要使用电脑做作业，曾到科技市场花了3000多块钱买了一台笔记本电脑。
小冯告诉记者，他们宿舍4个人现在用的电脑都是笔记本，携带方便。“之前用的都是XP系统，不是说XP系统下岗了吗，我们就想着到科技市场去换下系统，但是一问装个系统得20块钱，我们嫌贵，就自己在网上下了个win7的系统挨个儿装了装。”
“感觉科技市场就跟一个大的杂货市场一样，东西确实挺丰富、挺齐全的，但我不爱到那里买东西，买小的电子配件比如鼠标、键盘啥的，基本上点点鼠标就网购来了，跑到科技市场购买的话，环境乱，还得搞价钱，太麻烦。”

     [04.12]Perl，恐将消失的计算机语言 


根据Tiobe的程序语言指数排行榜，Perl语言流行度正处于历史最低点：一年前Perl排在第9位，一年后的今天排在第13位。

自2006年起Perl语言流行度就呈现下降趋势。Tiobe的总经理Paul Jansen说，在2006年左右，许多工程师所用的脚本语言从Perl变为了Python和Ruby。Perl衰落的原因可能包括语法异乎寻常，开发者会被搞混，而Python和Ruby能更好的支持Web开发。
Perl最初的设计者为拉里・沃尔（Larry Wall），他于1987年12月18日发表。Perl借取了C、sed、awk、shell scripting以及很多其他程序语言的特性。其中最重要的特性是它内部集成了正则表达式的功能，以及巨大的第三方代码库CPAN。简而言之，Perl象C一样强大，象awk、sed等脚本描述语言一样方便，被称之为“一种拥有各种语言功能的梦幻脚本语言”、“Unix中的王牌工具”。Perl一般被称为“实用报表提取语言”（Practical Extraction and Report Language），你也可能看到“perl”，所有的字母都是小写的。一般，“Perl”，有大写的P，是指语言本身，而“perl”，小写的p，是指程序运行的解释器。

     [04.12]马蔚华谈互联网金融监管红线：余额宝根本不用管 

4月8日-11日，博鳌亚洲论坛在中国海南省博鳌召开。在“互联网・金融：通往理性繁荣”分论坛中，永隆银行有限公司董事长、前招商银行行长马蔚华在谈到 互联网金融的监管红线时表示，“最近出现个别的P2P倒闭，也是损失了消费者利益，这是要管的。至于余额宝那根本不用管。”马蔚华表示，“看法是有，我想首先要研究一下互联网金融的兴起对监管的挑战，我想对货币政策和金融监管都有挑战，我觉得互联网金融的兴起对货币政策有非常大的影响，可以从货币的供给和需求两方面。互联网金融毫无疑问会提高资金的流动性，提高支付的速度，这样可以增加货币成数，减少现金漏洞，而且可以降低M2的现金和国际存款的比重，这就是内生的促进了货币增量。从需求方面，互联网金融效益高使交易非常快，减少了交易对货币的供给的需求，另外资金变现的能力也减少了投机性方面的货币需求，从供给和消费两方面，这对你原有的货币供应量肯定有非常大的影响。再就是你刚才讲到的互联网货币，比如Q币，1块钱的Q币用1元钱买，存款总量没增，但是创造了一个货币。这三方面可能都会在央行制定货币政策的时候要考虑控制基础货币的增长。”“金融监管的影响，过去传统思维就是我们在线下、物理网点考虑，比如说信用卡，我必须当面授信、我必须面签，我必须研究你能不能还款，现在虚拟空间发卡，商业银行没底、监管机构更没底。现在香港监管当局必须要求你说清录音，出去回来再买一个产品还要重新录音说清楚，这就为了保护消费者利益，确定事实，怕你在销售产品的时候隐瞒了他的风险。网络能不能解决这个问题？最近叫停了虚拟信用卡和二维码，我觉得这件事情如果从传统的金融观点讲，确实风险很大，心里没底。如果互联网思维想象空间很大，我不相信解决不了这些问题。但是在没有解决这些问题之前，谨慎一点也不是坏事。再加上互联网本身的特点，我上礼拜去一个国家，现在飞机买票都是完全电子化，偏偏很大的机场电子设备瘫痪了，结果我们一个、一个手工处理，原来一辆分钟就办，排了三个小时，你不可以设想网络出了问题是多么大的灾难。所有的客户、所有的资料、所有认证的枝节都在网上，如果预见黑客和网络破坏也是很大的风险。“监管应该重点维护网络安全，维护网络安全就是信息安全、消费者利益，这对现代监管当局是很大的挑战。像现代监管当局重视反洗钱，它上面留下的只是交易记录，没有真实影像，大额交易都需要相应的监管措施控制。互联网金融提出很多新的课题，这些课题不是不能解决，但是现在这阶段，我觉得监管的态度，就像小孩成长一样，要把不良嗜好去掉，让他健康成长，不要耍脾气，让他健康成长，这需要培养，需要呵护。中国有句话叫是玉烧三天满，对互联网不要一下子取缔、一下子制止，要呵护、观察它。”“二是不要一刀切的监管，世界各国也一样，有些国家，像澳大利亚和英国对互联网就非常宽松，而美国，因为次贷危机，制订了很多监管，还是比较严。中国不能一刀切，比如P2P出了一些问题，美国的P2P是提供平台，刚才谢平讲了，平台上自动交易，或者一对一、多对多都可以，但是平台要搞资金池、要搞担保公司，我觉得这个就触动了红线，最近出现个别的P2P倒闭，也是损失了消费者利益，这是要管的。”“至于余额宝那根本不用管，因为余额宝的出现是因为利率没有市场化，所以它自然的吸收银行的存款。有两条底线和五大原则，监管底线不能随便触碰，不能随便吸收公众存款，二是不能非法集资，这两个底线是监管当局掌握的。因为这条都涉及公众利益，会引发消费者损失和社会不稳定。”“中国关于互联网金融的舆论氛围比全世界任何地方都浓，因为我前两天开两会下来，所有的问题都是互联网金融，这是大家对新事物的感悟和关注是值得兴奋的。我觉得说阿里巴巴炒作没听说过，互联网金融对利率市场化肯定是倒逼和促进，以余额宝为例，余额宝确实分流传统银行很多的活期存款，银行的活期存款还是管制的，前提是你没放开，银行心里也是如焚如火，银行呼吁放开利率比以往任何时候强。因为存款利率不放开，1月份银行存款减少了8000亿，不都是余额宝吸引区了，因为银行体系外有很多、很多的金融的市场，包括PE、民间借贷、活期理财都比银行高，脱媒是必然的。有了余额宝以后，你需要花更高的利益把货币资金从同业存款吸收回来，这银行一想，我傻？我自己把货币利率放开就行了，所以倒逼银行的改革，对利率市场化是促进。”马蔚华最后说道。

     [04.12][科技不怕问]为何日本不像国内这样推限量套餐 

NTT Docomo 这种“通话定额收费制度”总价为5900日元，其中有2700日元（按当前汇率计算合人民币164.7元），只是智能手机通话套餐，3200日元（约合人民币195.2元）为数据通信费用。除此之外，DOCOMO还将首次引入与家人共享数据流量的计划。日本比中国要早8年用上3G，且3G网络信号覆盖的非常充分。从市场角度来说，日本属于典型的饱和市场。大运营商与小运营商之间价格战打的厉害。但大运营商是不适合打价格战的。大运营商因为市场份额高，继续扩大市场份额的难度很大，所以一旦发生价格战，就会直接导致毛利下滑，进而影响总收入。而小运营商因为可以通过价格战抢占大运营商的市场份额，虽然毛利下滑但是总收入上升，所以适合发动价格战。语音战转型为流量战价格战最立竿见影的益处是抢占市场份额，哪怕毛利很低。而iPhone是抢占市场的另一个杀手锏，曾经NTT Docomo一度不卖iPhone，因此丧失了很多年轻人群市场。但现在也开卖了。也就是说，无论是NTT Docomo还是软银，竞争差异无差别化了。随着4G网络发展下，移动应用生态的壮大，都迫使运营商萌发“去电信化”的思潮。要知道在日本是可以携号转网的，日本运营商非常注重的一个数据是“离网率”，即每月有多少用户跳转到其他网络，还剩多少留存用户。去年起NTT Docomo开始通过丰富的移动应用，通过更多优惠价格政策，把用户留在网内。比如打造“智能生活”概念，推出包括媒体内容服务、M2M服务、环境生态服务、医疗服务、金融服务、安全服务等。这就增大了数据流量的业务占比。这就导致了数据流量业务收入，会比语音通话业务收入占比高。从用户层面来说，虽然使用了定额语音通话+流量不限的套餐服务，但流量一旦超过带宽极限，就会自动降为3G网络。当然日本3G与4G网速差别不大。但价格战也是有瓶颈的。如去年下半年，日本运营商KDDI实行了一项4G促销计划：只要从软银转网的用户，不仅可以免费获赠iPhone 5，还可享受套餐优惠，每月通话+LTE流量无限制，大约4000多日元。国内4G不合适的玩法从语音通话的价格战到数据流量的价格战，日本大运营商与小运营商还能坚持多久，尚属未知。但这种玩法并不适用于国内。国内电信巨头都是大型国企，3G市场空间还未饱和仍有待开发，且价格战即使有不会白热化。但要实现4G套餐资费合理、透明，各家运营商也需要在“去电信化”的思路上走与日本截然不同的路数。

马航失联客机MH370的搜救到了最有希望，也是最艰难的阶段。从4月5日中国首次发现脉冲信号开始，各国在印度洋海域的搜救重点已经从飞机残骸转向了黑匣子。11日中午，澳大利亚总理托尼&#8226阿博特宣布已经能将马航失联客机的黑匣子锁定在几公里的范围内。虽然没有明确说明黑匣子的位置，但是他表态称搜寻人员对近日侦测到的信号来自失联客机黑匣子有信心。但是搜索面临的困难也非常巨大。中国国家海洋预报台专家说，这几天以来发现脉冲信号的区域是海底的海山区，山顶距离海面大约1800米，附近分布了海沟和海盆，海沟接近6000米深，从海山到海沟落差超过4000米，整个海底地形复杂，给搜寻工作带来很大困难。MH370黑匣子电池电量理论上只能持续30天，目前黑匣子上的水下声纳信标（Underwater Acoustic Beacons，简称水下信标）发射信号已经超过这一时间。对黑匣子的搜索正变得日益紧迫。那么应当如何对海底的黑匣子进行定位和追踪？可以用哪些技术手段？拖曳式声纳定位仪（拖鱼）等黑匣子定位装置第一步需要海面舰艇通过接收黑匣子发出的声波信号来尽可能地精确锁定黑匣子的位置。首先探测到黑匣子信号的是中国上海海事局的“海巡01号”。这艘搜救船上配备了美国Teledyne Benthos生产的黑匣子定位仪，最大可探测5000米水深。海巡01号具有侧扫声呐，可使用100米左右的缆绳将其放入海中，对黑匣子进行探测和定位。此后澳大利亚军舰“海盾号”也探测到了黑匣子信号，其探测深度为6000米水深。“海盾号”携带的“拖鱼”装置内部带有声波接收器，可通过在海面上缓慢拖行来探测黑匣子信号。为了能够让定位仪接收到黑匣子所释放出信号，“海盾号”只能以每小时三海里的速度进行搜索。截至10日晚间“海盾号”共5次探测到了脉冲信号。也正是通过多次信号探测，澳方才能够将黑匣子的大致位置进行锁定。不过这种探测装置会受到海况条件以及海洋生物的影响。据悉，为了避免海水涌动对信号探测形成影响，澳大利亚方面已经开始限制与“海盾号”在同一片水域内搜寻的舰船数量。水下航行器等水下探测设备如果对黑匣子有了比较精准的定位，可以再通过水下航行器（Autonomous Underwater Vehicles，简称AUVs）进行搜索。目前正在搜索黑匣子的“海盾号”，除了“拖鱼”设备，还携带了“Bluefin-21（蓝鳍金枪鱼）”，这是世界上最小的AUVs设备，由美国蓝鳍机器人公司（Bluefin Robotics Corporation）开发，状似鱼雷，装备摄影机与声纳，可潜入海底4500公尺处续航25小时。这款设备可以拍摄海底状况，以供工作人员进行图像观察和分析。目前澳大利亚方面尚未动用这款设备，原因是担心与正在工作的“拖鱼”装置互相干扰。澳方称待黑匣子彻底停止发射信号后，才会动用“蓝鳍金枪鱼”。这款设备依靠声呐在海底扫描，运动速度很慢，一天只能搜索相当于足球场大小的区域，搜寻一平方公里可能需要一个月。同一时间内，“拖鱼”的搜寻面积相当于“蓝鳍金枪鱼”的6倍，因此在仍然能够探测到黑匣子的情况下，目前还主要依靠“拖鱼”来进行黑匣子的定位工作。深海潜航器载人深海潜航设备也可以用于MH370的追踪和定位，中国的“蛟龙号”就是其中之一。中国“蛟龙号”载人潜水器是世界最深潜水设备之一，其最大下潜深度可以达到7000米。并且配备有高分辨率的声呐设备。蛟龙号深潜器副总设计师近日表示，只要失联客机定位精确，蛟龙号可以下海参与现场评估、搜救指挥甚至打捞工作。蛟龙号长处在于定点作业，非大范围搜寻。蛟龙号声纳设备工作范围只有数公里，由于航程限制，载人深潜器无法像无人深潜器那样在深海长时间作业。核潜艇等潜艇设备由于黑匣子电量有限，海面搜索受到海况情况影响较大，前期利用核潜艇快速扫描目标水域，对于搜索黑匣子非常关键。相对于“拖鱼”和“蓝鳍金枪鱼”，核潜艇在海底搜索上性能更加突出。核动力潜艇具备性能卓越的被动声纳系统，可感知黑匣子发出的信号。而且核潜艇可以在更高的航速下使用，搜索效率会大大提高。核潜艇在水下航行过程中，受水面噪音的干扰很小，而且续航时间较长，更适合水下搜索。2009年法航447航班失事坠毁于大西洋，确认失事海域之后，法国海军就曾派出“绿宝石”号核动力攻击型潜艇前往搜索。据BBC网站报道，英国国防部4月1日宣布，英国皇家海军特拉法加号核潜艇（Trafalgar class submarine）已抵达南印度洋，协助搜索马航MH370客机的黑匣子。除核潜艇外，一些深水潜艇也可以有效利用起来。在法航447航班搜索过程中，有三艘里莫斯6000型潜艇参与。这种潜艇是当时世界上最先进的水下探测设备，它可以在海底进行20小时的地毯式搜索，重新浮上海面，传回海底情况的声纳图像。其他辅助设备确定黑匣子等失联客机设备，一方面需要信号探测，另一方面也需要观察海洋状况，例如洋流方向，来确定飞机残骸的运动方向。可以使用海平面测量卫星，利用海洋热成像来观察洋流详情。除洋流状况之外，海底地形的测绘也必不可少。英国目前已经派出“回声号”军用测量船用于进行海底测绘。深海搜索非常复杂，法航447空难发生后，用了近两年的时间才发现客机的水下残骸。我们在利用各种高科技手段搜索海底客机的同时，更需要做的是反思应答器被人为关闭、黑匣子数据实时传输等客机失联前期问题，以防止类似悲剧的发生。

     [04.12]商务部官员：限额支付影响网购便捷性 正在协调 

正在此间举行的中国（义乌）世界电子商务大会上，商务部电子商务和信息化司副司长聂林海表示，电子商务目前面临包括限额支付在内的几大制约。聂林海说，电子商务经过高速增长后，目前面临一些发展制约和瓶颈。例如在互联网金融方面，关于限额支付，商务部正在跟相关部门沟通协调，“政府的一个重要任务是促消费，但这个限额会影响消费者在网上购物的便捷性。”一个月前，央行向第三方支付企业下发《支付机构网络支付业务管理办法》和《手机支付业务发展指导意见》的征求意见稿，其中的核心内容是对个人支付账户的转账、消费额度设限，此举引发了市场反响。此外，聂林海表示，电子商务所面临的制约还体现在，地区行业、企业界发展不平衡，拉大了中西部的发展差距；人才缺乏致使“挖人”现象普遍，导致人力成本上升。与此同时，市场监管服务水平达不到市场需求，这也是制约电子商务有序发展的重要原因。一些地方政府以传统的理念管理电子商务，管理部门繁多，降低了交易的效率。11日开幕的中国（义乌）世界电子商务大会由商务部、工信部指导，浙江省政府主办。为期两天的大会吸引了来自国内外的电商企业和近万名网商参加。

昨日，高德正式宣布与阿里巴巴达成确定性的收购协议，收购剩余的72%的高德公司股份。收购交易最终敲定 估值15亿美元2013年5月10日，阿里巴巴集团以2.94亿美元战略投资高德，持有约28%的股份。同时，阿里巴巴董事局执行副主席蔡崇信和阿里巴巴无线事业部总裁吴泳铭担任高德公司董事。阿里并不满足于持有28%的股份。今年2月10日，阿里巴巴集团表示，已向高德控股提出收购建议书，拟以11亿美元现金，收购剩余的72%的高德公司股份。高德公司当时表示，公司董事会已经收到该要约，并计划成立一个独立委员会，对该提议进行评估。今日，双方正式达成收购协议。根据协议内容，在收购完成时，高德股东将获得每股普通股5.25美元或每股美国存托股份（“ADS”）21.00美元的收购对价。该收购价格比2014年2月7日（即公司在2014年2月10日宣布其已收到阿里巴巴私有化建议之前的最后一个交易日）高德股票纳斯达克收盘价每股ADS 16.54美元的价格溢价了27.0%，较公司30和60个交易日量加权平均价的溢价分别是38.5％和39.8%。交易目前预期将在2014年第三季度完成，对高德在完全转股且摊薄的基础上的股权价值估值约为15亿美元。阿里收购高德为O2O加码这项收购并不意外。对于阿里巴巴来说，收购高德地图将增加其发展O2O的砝码。阿里巴巴在本地生活服务领域曾进行多项投资，如美团网、丁丁网。阿里巴巴今年的战略是ALL IN移动电商，手机淘宝定位于移动商业的入口。在这一构想中，精准、有效的地图是不可或缺的一环。地图是大型互联网公司的战略产品。谷歌(微博)拥有谷歌地图，苹果为了摆脱谷歌地图，自己研发出地图产品。在国内，腾讯地图、百度地图皆形成了一定的规模。但阿里投资高德之前，尚没有一款可以与竞争对手抗衡的地图产品。虽然阿里投资了丁丁网，但丁丁网的重要资产是丁丁优惠券，丁丁地图并没有做大；2012年10月份，淘宝本地生活曾推出“地图搜”用户可以通过地图搜索优惠和商户，但未能形成影响力。在地图产品中，高德拥有甲级导航电子地图测绘资质和甲级航空摄影测绘资质，拥有优质底层地图数据，是阿里的最优选择。互联网评论人士洪波(微博)认为，阿里巴巴集团收购高德后，可为阿里巴巴的O2O加码。“阿里巴巴除了已经做的很成功的、几乎已经到达顶点的电商业务之外，留给投资者的想象空间并不多。而微信支付、微信O2O的成长速度非常快，会进一步让投资者看淡阿里巴巴的未来。因此，阿里巴巴需要更多具有想象空间的业务，O2O是重要领域。”高德借阿里向B2C转型作为一家B2B起家的公司，高德选择了逐步向B2C转型，高德软件CEO成从武曾概括为从数据到平台到入口的“三级跳”：第一跳是地图数据。第二跳是开放合作。第三跳是用户端增长。高德去年提出了客户端要突破2亿的目标，把产品真正打造成一个移动互联网的产品，从纯工具性的产品演化成生活服务的产品。无论是高德如何试图引入互联网公司的人才，但因为B2B的基因根基强大，真正实现转型，需要借助互联网公司的力量。高德CEO成从武去年5月份接受腾讯科技采访时曾表示，过去地图是工具化产品，实际上地图可以承载更多内容，这是它的特点和优势。基于地图打造一站式的服务是顺理成章的。“但如果自建支付、电商，成本太高。因此通过整合、开放的方式，接入阿里的资源。”基于双方大数据分析，成从武当时设想了未来可能出现的应用场景：一是根据用户过去本地消费的行为，推送个性化服务。例如，你设定好路径规划导航，在“左拐右拐”的指引中，突然告诉你附近有一个你喜欢的咖啡店，你会觉得很惊喜。当然，用户也可自己设定，选择开启或关闭，达到双向互动。二是基于地理位置，将线上线下打通。用户可以通过高德地图搜索商户、商品名称，将附近有这件商品的店面展现出来，也可能通过电子支付，送货到家。文化整合存挑战收购完成后，将有助于双方数据的对接与整合。在这项收购之前，高德已与阿里集团旗下淘点点整合，淘点点采用高德地图提供的位置服务，双方还将全面打通商户数据。分析人士认为，收购后的挑战在于文化整合。在阿里集团内部，“阿里味”是阿里人特别强调的一种特质。作为外来者，被收购后如何融入阿里集团，这是除业务整合外一个更大的挑战。对于文化融合问题，成从武去年曾对腾讯科技表示，与什么样的公司合作都存挑战。“过去我跟马总（马云）见面不是很多，但一见面就有缘份。”透露出对双方整合的信心。

很难想象作为前思科中国总裁，林正刚在微博、QQ群里和粉丝们打成一片，分享交流管理经验。从思科退休后，他给自己重新定位为刚逸领导力公司CEO、企业教练和管理专家，将自己30多年的管理经验传递给更多企业和创业者，并致力于为他们打造信息化沟通平台。近日，腾讯科技独家专访了林正刚。谈及创业者，他表示我国中小企业最欠缺的是企业运营基本功和社会经验，这也是目前很多企业面临的普遍问题。生在香港，学在加拿大，海外工作十年，而后又在中国大陆市场工作二十余年，先后供职于DEC及思科，这样的经历，使得林正刚对中西方企业差异有更深刻的感受。在他看来，国内外创业者最大的区别在于，国外创业者通常是在企业中沉淀一段时间，拥有一定经验后才去创业，而我国创业者比较心急，缺乏社会经验和基本功，甚至很多人大学时代就一边逃课一边创业，这样意义并不大。林正刚一直在强调企业运营的基本功问题，“这是我国中小企业面临的核心问题，基本功不够，即便成功了，也没有可复制的借鉴意义。”如何将其国际化企业管理经验有效作用于中小企业，是一个很现实的问题。对此，林正刚直言，他所提炼的企业运营框架是最基本的原理，目的是要打好基础，培养正确的思维。“我不会教他们技巧，因为我的技巧他们未必适用。所以我只谈思路，也就是通过理论框架，回到最原始的基础问题上。”林正刚说。什么是原始的基础问题呢？林正刚总结了一套企业运营的路线图。在这套路线图里，有着很多看似很大的词儿。比如价值观、核心能力、沟通与反思等。但这往往是中小企业在从小到大过程中最为忽视的事情。林正刚认为，很多中小企业都面临发展的瓶颈，其根本原因在于没有梳理清楚公司的运营逻辑，以为价值观等只是喊喊口号。但实际上并不如此。在最近上海的一个课堂上，林正刚列举了价值与核心能力之间的辩证关系。他认为，对企业而言，商品有价值，不代表企业拥有核心能力，比如空气有价值，但生产空气的企业是否有核心能力呢？显然没有。“很多企业会将这两者的关系搞混。现在依靠有价值的产品成功了，但实际上并没有建立自己的核心能力，这必然会导致长远的发展受限，甚至走不下去。”林正刚解释道。林正刚希望自己多年的跨国管理经验能够作用于更广泛的企业家人群。在未来，他还要将培训课程普及给大学在校生。因为目前其开发的课程成本较高，能够负担起的人还是少数，他希望将课程通过大学生平台传递给经济能力较弱、但可复制度高的创业群里，帮助他们摆正价值观，少走弯路。早在2010年，林正刚就曾就退休一事说过：“‘退休’不存在我的思维里面，只是我会从职业经理人的角色转换到创业与公益上面，过去我只能服务少数公司，以后我希望能发挥我的经验来帮更多的企业。”如今看来，他并未食言。以下为采访实录：外企和民企的最大差别：管理机制和管理文化腾讯科技：我发现来参加您培训的企业基本上都是中小民企，而您以前一直都是在为世界500强企业服务，您觉得外企和民企它们最大的差别在哪里？林正刚：2010年退休后，我一直都在做民企的企业教练和管理培训，对他们的认识的确深入了很多，就我这段时间的观察，我认为最大的差别是外企的管理效率比我们民企高。外企效率高的原因是他们有比较成熟的管理机制，凡事都有规可循。另外还有文化差异，外国人的法治观念比较到位，在外企，当企业定下了规矩，就是这个规矩不太合理，在没修改之前还是要照规矩执行。而在民企，这种做法往往被看成是傻，不够灵活，不懂国情，所以就出现“上有政策，下有对策”的情况，这样一来，企业会产生很多内耗，效率就提不上去。腾讯科技：那是什么原因造成这样的差别呢？林正刚：在国外，经过100多年的发展，它的市场环境已经比较成熟，管理机制就是这样一点点建立起来的，而我们的市场经济才30多年，民企的时间还很短，管理机制的积累是需要时间的。另外还有一个原因，我们的民企老板不太喜欢搞机制，认为这种是不灵活的做法，没有机制，那就是个人说了算，如果永远都是小企业，这种做法还能接受，但现在很多民企都想冲出国际，要做大做强，这就带来很大的挑战了。企业的所有问题都是一把手引出来的：改变要从最上面开始腾讯科技：来参加您的培训的企业大概是一个什么样状态和规模呢？林正刚：目前来说，我培训的企业大部分都是已经过了生存期，前期发展得还不错，但同时也遇到了管理瓶颈，苦于企业效率提升不上去，然后来找到我培训。就规模来讲，这些企业的量级大概是从几千万到几个亿，不会太大，但也不会太小，因为太小是没有足够的精力和财力去学，我的时间成本高，能承担起这个价格的人还是少数，所以我也正在想用什么手段把这个课程通过一种平台传递给暂时付不起钱的人。腾讯科技：您在选择企业和培训上有什么特别要求吗？林正刚：对一个企业来说，企业出了问题，源头一定在CEO，所以要想真正帮到企业、改变企业，CEO必须要能承认问题是出在自己身上，这个是很有挑战的。我跟这些培训的老板说      [04.12]一男子利用网络漏洞虚充1200余万话费 被判5年 

小伙黄某利用系统漏洞，半个月内竟非法充值话费1200余万元。近日，重庆市第五中级人民法院对该案进行终审宣判，黄某因犯盗窃罪被判处有期徒刑五年，退赔30余万元，并处罚金2万元。法院审理查明，黄某自2006年起在淘宝网上经营SIM卡及话费业务。2011年9月底，黄某在某移动公司官方网站上，将话费通过“家庭账户”业务转账到家庭账户的时候，无意中发现其官网上的“家庭账户”业务存在的系统漏洞。黄某遂利用该漏洞，进行重复转账操作，使其持有的SIM卡中话费共计虚增1210.2448万元，黄某将其中41.091万元话费通过“家庭账户”业务转账到接收话费手机号码，并经银行转账、淘宝网支付宝支付资金或现金交易等方式卖给罗某、朱某、吴某、梁某等人。2011年10月中旬，某移动公司重庆有限公司发现有客户通过“家庭账户”业务转账系统恶意套取话费后，于当月下旬向公安机关报案。2011年10月31日，公安机关将黄某抓获，并在其家中扣押了270张SIM卡，移动公司从其中的118张SIM卡中冻结话费1167.9738万元，另冻结了黄某销售给罗某、朱某、吴某的SIM卡中话费共计10.740678万元。重庆五中院认为，黄某以非法占有为目的，利用其发现的漏洞，非法使其持有的手机卡话费成倍增加，并将虚增的手机话费销售给他人，造成移动公司电信资费实际损失共计30.3万余元，依照最高人民法院《关于审理扰乱电信市场管理秩序案件具体应用法律若干问题的解释》，其行为已构成盗窃罪，且数额巨大。因此，黄某的行为应以盗窃罪定罪处罚。(陈小康陈铜)
